# Prompt Improver Examples

## Example 1

Original prompt:

```text
explain blockchain
```

Improved prompt:

```text
Explain blockchain to undergraduate students who are new to the topic. Use simple language, define key terms, and include one concrete real-world example. Keep the explanation under 500 words and end with a short summary of why blockchain matters.
```

Improvements made:
- added audience
- added scope and length
- added request for definitions and an example

## Example 2

Original prompt:

```text
write a report about climate change
```

Improved prompt:

```text
Write a short briefing on climate change for city policymakers. Summarize the main causes, the most relevant local risks, and three practical policy responses. Use clear section headings and keep the tone professional and concise.
```

Improvements made:
- added audience
- narrowed the scope
- specified output structure and tone

## Example 3

Original prompt:

```text
help me make my app better
```

Improved prompt:

```text
Review my app idea and suggest improvements in three categories: usability, core features, and differentiation from competitors. For each suggestion, explain why it matters and how difficult it would be to implement. If important context is missing, list the questions you need answered first.
```

Improvements made:
- converted a vague request into a concrete review task
- specified categories
- requested rationale and implementation difficulty
- added a fallback for missing context
